Features: 
-8-key input (buttons K1–K8) for triggering notes
-Rotary encoder for octave shifting (±3 octaves)
-Joystick Y-axis for real-time pitch bending
-Transposition button (−6 to +6 semitones)
-Six selectable modes:
-Major, Minor, Lydian, Mixolydian, Dorian, Phrygian
-Built-in tone generation through the Arduino’s tone() function
-Serial output for mode monitoring and debugging

Requirements:
-Arduino IDE
-Arduino UNO R3 (or compatible)
-Piezo speaker or buzzer
-8 push buttons
-Rotary encoder with push button
-2-axis joystick module
